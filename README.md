# 🌿 Rhythra – Rhythm-Aware Wellness Companion

Rhythra is a modern wellness and productivity platform designed to help users live in alignment with their **energy, habits, emotions, and productivity rhythms**.

The application combines habit tracking, mood analysis, AI insights, and gamification into an intuitive dashboard that encourages sustainable personal growth.

This repository branch contains the **Rhythra Pro UI upgrade**, introducing a redesigned interface and several intelligent features.

---

# 🚀 New Features in This Branch

## 🎨 Pro UI Dashboard

* Clean **SaaS-style dashboard layout**
* **3D interactive cards**
* **Apple-style animated wellness rings**
* Improved spacing, alignment, and responsiveness

## 🤖 AI Wellness Engine

* **Energy prediction system** based on mood trends
* **AI insights** recommending wellness actions
* Built-in **AI wellness chatbot**

## 🔁 Habit Tracking System

* Habit creation and completion tracking
* **Habit streak system**
* **Celebration fireworks animation** for streak completion
* **GitHub-style habit heatmap calendar**

## 📊 Mood Analytics

* Daily mood logging
* **Visual mood trend chart**
* Mood data used for AI insights and wellness score

## 🧠 Gamified Growth System

* XP earned from habits, mood logs, and tasks
* Level progression:

| XP        | Level     |
| --------- | --------- |
| 0 – 49    | Seed 🌱   |
| 50 – 149  | Sprout 🌿 |
| 150 – 299 | Bloom 🌸  |
| 300 – 599 | Zenith 🌟 |
| 600+      | Master 🌌 |

## 🌈 Theme System

Supports **12 UI themes** including:

* Blossom
* Lavender
* Peach
* Rose
* Mint
* Dark
* Ocean
* Navy
* Forest
* Grey
* Indigo
* Default

Themes are saved using **localStorage**.

# ▶️ How to Run

1. Clone the repository

```
git clone <repository-url>
```

2. Open the project folder

3. Run the application by opening:

```
login.html
```

4. Enter a username to start using Rhythra.

No backend or installation required.
